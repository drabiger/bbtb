Info What happens if I create a user account?

The basic idea is that BBTB stores only a minimum of your personal data. The data you create in BBTB will be protected.

When creating a BBTB user account, BBTB will ask for permission to access basic profile information from your Google account. Note that only your email address will be stored. BBTB will not use your real name from Google. Instead, you will be asked for a display name that will appear next to content that you would like to be visible to other BBTB users.

BBTB

Blood Bowl Tactics Boards

View project on GitHub

About

BBTB will become a web site for Blood Bowl fanatics who like to plan their Blood Bowl matches using tactics boards. A tactic board lets you visualize a situation in the game, displaying the board and a number of players distributed over the field.

Current Status

Far from being presentable. Don't expect any public access before 2016.
